# UI Subsystem Architecture & Contracts

This document specifies the design system contracts for layouts, themes, and display states in PomoGo.

---

## 1. The `DisplayState` Data Contract

The `DisplayState` struct in [layout.go](file:///home/ibnuafdel/projects/Go/pomogo/internal/render/layout.go) is a pure, self-contained snapshot of the TUI's state at any given tick. Renders/layouts consume this struct, making them completely decoupled from Bubble Tea logic.

```go
type DisplayState struct {
	ModeLabel        string
	Project          string
	Task             string
	PhaseKind        timer.SessionPhase
	SegmentRemaining time.Duration
	BlockRemaining   time.Duration
	Progress         float64 // 0.0 to 1.0
	SegmentIndex     int
	SegmentCount     int
	Paused           bool
	Running          bool
	Idle             bool
	StatusMessage    string
	HintsVisibility  bool
	ThemeName        string
	LayoutName       string
	Zen              bool
	GitBranch        string
	TmuxSession      string
}
```

## 2. Layouts Contract

Layouts are pure functions implementing the `Layout` type:
```go
type Layout func(ds DisplayState, th *theme.Theme, f Frame) string
```

### Layout Registry
All layouts must register themselves during initialization with `render.Registry`:
```go
func init() {
	Registry["classic"] = LayoutSpec{
		Layout:    Classic,
		MinWidth:  40,
		MinHeight: 10,
	}
}
```

## 3. Theme Palette Slots

The `Theme` struct contains 12 semantic color roles mapping to Lip Gloss styles:
1.  **Work**: Theme color for work sessions (e.g. Red/Coral).
2.  **Break**: Theme color for short break sessions (e.g. Green).
3.  **LongBreak**: Theme color for long break sessions (e.g. Teal).
4.  **Idle**: Theme color for idle/menu state.
5.  **Accent**: Border/high-focus elements color.
6.  **Muted**: Lower-focus labels/helper text.
7.  **Text**: Body text.
8.  **Background**: General window background.
9.  **Border**: Structural border color.
10. **ProgressFill**: Active filled sections of progress bars.
11. **ProgressTrack**: Unfilled track background.
12. **Subtle**: Quiet/ambient decorations.
